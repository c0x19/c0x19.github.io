环境变量

0x01 什么是环境变量

程序和脚本通过环境变量来获取

  • 系统信息
  • 存储临时数据
  • 配置信息

环境变量分为两种：

  • 全局变量：对于shell及它的子shell都可见
  • 局部变量：只shell自己可见

查看环境变量

  • 查看全部全局变量
    • printenv
    • env
    • set
      • 可以查看全局变量、局部变量、用户定义的变量
  • 查看单一环境变量的值
    • printevb HOME
    • echo $HOME

设置用户定义变量

  • 创建局部变量: my_var="hello world"
    • 用户定义的环境变量使用小写，区分系统的大写环境变量
    • 环境变量值中有空格要用双引号
  • 创建全局变量
    • my_var="hello world"
    • export my_var
  • 删除环境变量
    • unset my_var

0x02 PATH环境变量

环境变量中有一个PATH环境变量

  • 存储命令和程序查找的目录

  • 每个目录用冒号隔开

  • 方便程序在任意目录中快速执行，而不必进入其目录下才能运行

添加新目录

PATH=$PATH:/home/nik/test

通常运行一个程序需要这样：./test

我们可以将单点符加入PATH环境变量来方便运行程序

  • PATH=$PATH:.
  • test

0x03 持久化存储环境变量

在~/.bashrc文件中添加或修改环境变量即可

更新生效

source ~/.bashrc
